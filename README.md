# -SQL-Project_TASK6
Sales Trend Analysis Using Aggregations
This project analyzes monthly revenue and order volume using SQL aggregation techniques on an online sales dataset.
The repository includes a complete MySQL script for database creation, dataset insertion, and monthly sales trend analysis using SUM() and COUNT(DISTINCT ...).
Key insights include identifying high-performing months, revenue patterns, and order frequency trends.
The project is useful for learning SQL grouping, filtering, and time-based analysis.
